# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-16
### Added
- Terminal Matrix rain animation with multiple color sets.
- Centered 3x5 text rendering with a border (built-in font).
- Background scroll speed control and listable color palettes.
- Quit controls: `q`, `Esc`, or `Ctrl+C`.
